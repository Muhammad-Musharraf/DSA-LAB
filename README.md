## 📖 About

Welcome to **DSA-LAB**, a repository containing my **Data Structures and Algorithms (DSA)** laboratory programs and practice questions implemented in **C++**.

This repository is created to document my learning journey as a **BS Computer Science** student. It includes fundamental data structures, searching and sorting algorithms, recursion, trees, graphs, and other important DSA concepts.
